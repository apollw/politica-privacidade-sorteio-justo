Política de Privacidade - Sorteio Justo

Última atualização: [Data Atual]

O aplicativo Sorteio Justo respeita a privacidade de seus usuários. Nossa política é simples: não coletamos, armazenamos ou compartilhamos nenhum dado pessoal dos usuários.

1. Dados Armazenados

Todas as informações inseridas ou geradas pelo aplicativo são armazenadas apenas no dispositivo do usuário.

2. Compartilhamento de Dados

Não compartilhamos nenhum dado com terceiros, pois o aplicativo não coleta nem transmite informações pessoais.

3. Segurança dos Dados no Dispositivo

Os dados armazenados localmente são protegidos pelas configurações de segurança do dispositivo do usuário.

4. Alterações nesta Política

Podemos atualizar esta Política de Privacidade ocasionalmente para refletir mudanças no aplicativo. As alterações serão comunicadas por meio de uma nova versão do aplicativo.
